A simple web app using NodeJs

1. npm init | to intialize the package json
2. npm install express | to install express depedencies
3. npm install redis | to install redis depedencies
4. add script in package.json "start": "node index.js" 
5. update index.js
6. create Docker file
7. create DockerCompose.yaml file
